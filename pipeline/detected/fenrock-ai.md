## Thesis
Back-office compliance is the hidden drag on every bank. AML teams run perpetually understaffed; case backlogs lead to regulatory fines; loan processing queues create customer attrition. Fenrock addresses this with an agentic overlay — not a rip-and-replace of legacy systems, but an intelligence layer on top. Your thesis includes the platform infrastructure for making the financial agent garden safe; Fenrock is a specialist agent sitting inside that garden, executing regulated workflows with traceable outputs. The audit-log framing is critical: in financial services, every agent action must be defensible to a regulator. Fenrock builds that in by design. The team's background in regulated data (healthcare API at scale, Apple privacy ML) maps cleanly to the GDPR/BSA/AML environment banks operate in.
