## Thesis
Overmind was archived in April 2026, reopening the AI Safety gap. Capsule addresses exactly the same need: supervised agent operation where the platform enforces controls between the moment an agent receives a prompt and the moment it acts. Unlike model-level AI safety (Mindgard 2★), Capsule is explicitly agent-level — monitoring multi-step execution, enforcing guardrails, preventing manipulation. The runtime gap (prompt → action window) is precisely where financial agents are most exposed. This is architecture you would want sitting in the stack for any external agent marketplace where agents touch real money, data, or compliance-sensitive workflows. "No proxies, gateways or SDKs required" means low friction adoption — critical for regulated institutions.
