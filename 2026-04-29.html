S-Front / S2 / S8 / Tax — Multi-Fit Specialist New Signal

Autonomous (ex-Paperspace founders, $15M pre-seed) — Multi-Specialist Wealth Agent

Autonomous Technologies Group emerged from stealth with a $15M pre-seed led by Garry Tan / Y Combinator, alongside Box Group and Collaborative Fund. Founded by Dillon Erb and Daniel Kobran — the Paperspace co-founders who sold to DigitalOcean for $111M. The product, called Autonomous, is positioned as an AI-native wealth strategist offering "billionaire-style playbooks" at 0% advisory fees. Crucially, it coordinates across 401(k), taxable accounts, mortgage, equity, and cash flow — and offers tax-aware portfolio management by working with individual stocks rather than index funds. This is the strongest dual-fit specialist agent the BG radar has surfaced: it touches S-Front, S2 Portfolio Construction, S8 Financial Planning and Tax Optimisation simultaneously. Geography: NYC + SF. Stage: pre-seed (perfect). Note the SEC predictive-analytics-rule risk applies — they will need RIA registration similar to Era to operate fiduciarily. New startup profile created: autonomous.

Agent Payment Rails (underrepresented) New Signal

Sponge (YC W26, ex-Stripe Crypto team) — Financial Infrastructure for Agents

Sponge is a YC W26 launch building financial infrastructure for the agent economy — agents hold funds and transact autonomously with other agents and businesses via bank accounts, cards, and crypto. The three-person founding team is entirely ex-Stripe Crypto (Eric Zhang, Jae Choi, Rishab Luthra — combined ~15 years at Stripe), and explicitly built Stripe's stablecoin financial accounts and core money movement systems. The product ships with wallets, gateway, agent skill, TypeScript SDK, MCP integration, and multi-chain support (Base, Solana, Tempo). This is the closest US analogue to Nava (5★) in the BG pipeline, with stronger founder pedigree but heavier crypto orientation — the multi-chain crypto component is a yellow flag for a regulated-brokerage app store. Geography: US. New startup profile created: sponge.

Agent Monetisation (underrepresented) New Signal

Orthogonal (YC W26) — API Marketplace for AI Agents

Orthogonal is a YC W26 launch offering "agentic payments for APIs" — a marketplace where agents access multiple APIs through a single key, with usage-based billing handled by the platform. This is a sub-shape adjacent to Paid AI and Paygentic in the Agent Monetisation category, but with a distinctive marketplace twist: rather than billing for the agent's own outputs, Orthogonal monetises the agent's tool usage. Maps to a sub-domain of Agent Monetisation that the thesis hadn't explicitly named. Note: not finserv-specific, which is a scoring drag (–10). New startup profile created: orthogonal.

Agent Payment Rails (voice-specific) New Signal

Maven (YC W26) — PCI-Compliant Voice Agent Payments

Maven is a YC W26 launch providing PCI-compliant card-payments infrastructure for voice AI agents — a payments API that lets a voice agent collect a payment mid-call. Disclosed pricing (pay-per-session, free tier), integrations with Stripe, Authorize.net, and Adyen. The thesis fit is narrower than Sponge or Orthogonal: voice-agent payments don't naturally translate to a regulated brokerage app store where the dominant interface is text/structured-intent rather than voice calls. Carded for tracking but scored low. New startup profile created: maven.
