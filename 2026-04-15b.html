Competitive Intelligence

eToro App Store — Launched April 14, 2026

eToro has launched a third-party developer marketplace with: (1) a builders portal giving developers structured access to APIs, agent skills, MCP server, CLI tooling, and docs; (2) Agent Portfolios where users allocate capital and connect their own AI agent via scoped API key ($200 minimum); (3) social trading extended from "copy a human" to "copy an AI agent." This is the first live implementation of the "app store for financial agents" model. No governance layer, no pre-trade compliance, no agent identity management. The infrastructure gap the thesis targets is real and visible.

Public.com Agentic Brokerage — Live Rollout

Public.com's agents now support stocks, options, and crypto. Users describe strategies in plain language ("If VIX hits 25, buy a put on the S&P 500") and agents handle monitoring, decision-making, and execution. Every agent runs within the authenticated brokerage environment. No external agent marketplace — all agents are built by Public.com internally. This confirms the thesis: consumer-facing agentic brokerage is live, but the external agent marketplace model hasn't been attempted by Public.com.

OpenAI Acquires Hiro Finance — Financial Planning Gap Narrows

OpenAI acquired Hiro Finance, an AI financial planning startup. This removes Hiro from the scoutable pipeline but signals that frontier model providers are moving into financial planning (S7). The financial planning agent gap is now addressed by a hyperscaler, not a startup — which may mean the opportunity for an independent startup in this category is compressed.

Structural Finding: The Specialist Agent Desert

This cycle's most important finding is negative: 8 of 9 specialist agent categories (S2–S9, excluding Tax Optimisation) have zero pipeline companies after intensive scouting. Web searches across portfolio construction, trade structuring, options strategy, execution quality, position monitoring, financial planning, behavioural coaching, and post-trade analysis turned up almost no seed-stage startups building these as agent-callable APIs.

This could mean: (1) These capabilities are too specialised to exist as standalone startups yet — they may emerge as features of larger platforms. (2) The search terms need further refinement. (3) These agents are being built inside incumbents (Vanguard's autonomous TLH, BlackRock's portfolio tools) rather than as startups. (4) The eToro App Store may catalyse startup creation in these categories as developers build for a live marketplace.

Implication for the thesis: The "specialist agent" half of the app store may require a different scouting strategy — partnering with quantitative teams, monitoring eToro's app store for developer activity, or tracking academic spinouts rather than traditional VC-backed startups.
